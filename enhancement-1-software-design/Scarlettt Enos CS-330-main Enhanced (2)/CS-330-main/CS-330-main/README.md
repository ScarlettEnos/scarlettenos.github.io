# CS 330 Portfolio

For this project, I started by thinking about the overall idea of the scene and then breaking it down into smaller parts I could work on one at a time.
I looked at the reference image and figured out which basic shapes I could use to build each object. Instead of trying to create everything all at once, 
I focused on one object, adjusted it until it looked right, and then moved on. As I continued building the scene, I kept tweaking the size, placement, and
rotation of each object until everything looked balanced and fit together well. I also kept my code organized by using reusable functions for things like 
transformations, materials, and textures so I wasn’t copying and pasting the same sections over and over. That made it much easier to fix things later without 
having to redo large parts of my code. A lot of my progress came from testing things out and making small changes, especially with lighting and camera movement,
since those took multiple tries to get right. Over time, I became more comfortable making careful adjustments instead of big changes all at once. This project 
definitely improved my patience and made me realize how much effort goes into even simple 3D scenes. Even though I don’t plan to work in computer graphics, 
it gave me a better appreciation for the digital media I use and enjoy every day.
